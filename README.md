# Ultra kort intro

Alt du skal bruge er Docker og Docker-compose.

Klon projektet til en ønsket mappe.

`cd grafana-influxdb`

`docker-compose up`

I din browser åbner du http://ip-dresse:3000

Så er du kørene:-)
